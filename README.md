## 🚀 How to Run Locally

1. **Clone the repository:**
   ```sh
   git clone https://github.com/rougepavan/porfolio1.git
   cd porfolio1
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:5173/`.

## 🌐 How to Host Online (Best Methods)

The best way to host this React + Vite site online for free is using **Vercel** or **Netlify**. Both platforms connect directly to GitHub, rebuild automatically when you push changes, and provide free HTTPS/SSL custom links.

### Method A: Hosting on Vercel (Recommended)
1. Sign up/log in to [Vercel](https://vercel.com/) using your **GitHub** account.
2. Click **"Add New"** > **"Project"** on the dashboard.
3. Import the **`porfolio1`** repository.
4. Keep all default configurations (Vercel automatically detects the Vite template and sets the build/output commands).
5. Click **"Deploy"**.
6. Your live link will be generated in under a minute!

### Method B: Hosting on Netlify
1. Sign up/log in to [Netlify](https://www.netlify.com/) using your **GitHub** account.
2. Select **"Add new site"** > **"Import an existing project"**.
3. Authenticate with **GitHub** and select the **`porfolio1`** repository.
4. Leave the settings at their defaults:
   * Build command: `npm run dev` or `npm run build`
   * Publish directory: `dist`
5. Click **"Deploy porfolio1"**.
6. Netlify will build the project and provide your live link!
